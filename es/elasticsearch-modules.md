# Elasticsearch功能模块。
## static
这些设置必须在节点级别（在elasticsearch.yml文件中）设置，或者在启动节点时在环境变量或命令行上设置。
必须在集群中的每个相关节点上设置它们。
## dynamic
可以使用[cluster-update-settings](https://www.elastic.co/guide/en/elasticsearch/reference/current/cluster-update-settings.html) API在活动集群上动态更新这些设置。
