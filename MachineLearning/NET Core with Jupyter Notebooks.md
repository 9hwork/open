1. 安装dotnet-try
```
dotnet tool install -g dotnet-try
```
2. 安装dotnet kernel
```
dotnet try jupyter install
```
查看安装列表
jupyter kernelspec list


3. 启动jupyter notebook
```
jupyter notebook
```
