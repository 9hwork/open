## Can I use 一款前端兼容性自查工具
https://caniuse.com/

npm  install -g caniuse-cmd

 注意：如果使用的是Windows系统，需要在Git的命令行执行上述命令。否则会报Git Not Found的错。

用法：

基本用法：caniuse  特性

如运行：caniuse websocket
caniuse 默认查看的是PC浏览器的兼容性，它也提供了查看在移动端的数据。
执行命令：caniuse -m websocket
